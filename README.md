# הבית שלנו

אתר זוגי קטן וחמים לניהול הבית: משימות, רשימת קניות ותזכורות במקום אחד.

## הרצה מקומית

1. התקן חבילות:

```bash
npm install
```

2. הפעל שרת מקומי:

```bash
npm start
```

3. פתח בדפדפן:

`http://localhost:5173`

## כתובת ציבורית

כרגע אין לאתר כתובת ציבורית אוטומטית מתוך המחשב הזה, אבל הוא מוכן לפרסום מהיר.

### אפשרות מומלצת: Vercel

1. היכנס ל-`https://vercel.com`
2. צור פרויקט חדש
3. העלה את התיקייה הזו או חבר ריפו GitHub
4. Vercel יזהה שזה אתר סטטי ויפרסם אותו
5. בסוף תקבל כתובת בסגנון:

`https://bayit-shelanu.vercel.app`

קובץ [vercel.json](c:/YanivAI/vercel.json) כבר נוסף כדי לשמור על פריסה פשוטה ונקייה.

### אפשרות פשוטה בלי Git

אם אתה רוצה דרך מהירה בלי ריפו:

1. פתח את `https://app.netlify.com/drop`
2. גרור לשם את כל תיקיית הפרויקט `c:\YanivAI`
3. תוך כמה שניות תקבל לינק ציבורי

## מה האתר כולל

- משימות לבית
- רשימת קניות משותפת
- תזכורות עם שעה
- שמירה ב-`localStorage` בדפדפן

## הערה חשובה

הנתונים נשמרים מקומית בכל דפדפן בנפרד. כלומר, אם אתה ובת הזוג שלך תפתחו את אותו לינק ממכשירים שונים, כל אחד יראה את הרשימות שהוא יצר אצלו, ולא סנכרון משותף אמיתי.

אם תרצה, השלב הבא שאני יכול לעשות הוא להכין לך גרסה עם סנכרון אמיתי אונליין, כך שלשניכם תהיה אותה רשימה בדיוק.
